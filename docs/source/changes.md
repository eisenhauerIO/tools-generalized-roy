# Changes

This is a record of all past `grmpy` releases and what went into them in reverse chronological order. We follow [semantic versioning](http://semver.org/) and all releases are available on [PyPI](https://pypi.python.org/pypi/grmpy).

## 1.0.0 - 2024

This is the initial release of the modernized `grmpy` package with:

- Modern Python 3.11+ support
- Hatch-based build system
- Updated documentation with MyST Markdown
- GitHub Actions CI/CD
